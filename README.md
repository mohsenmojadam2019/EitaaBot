ربات ایتا چه کاری انجام می‌دهد؟
1. در گروه یا کانالی که عضو است، پیام‌ها را می‌خواند
2. هر پیامی که می‌آید را در دیتابیس ذخیره می‌کند
3. بر اساس کلمات کلیدی، خبر را دسته‌بندی می‌کند (مثل سیاسی، ورزشی، اقتصادی)
4. می‌توانیم بعداً گزارش بگیریم

✅ قدم‌هایی که باید برداری:
مرحله	کار
1️⃣	در ایتا به ربات @BotFather برو و یک ربات جدید بساز (توکن بگیر)
2️⃣	دیتابیس را در phpMyAdmin بساز
3️⃣	فایل‌ها را به ترتیب کپی کن
4️⃣	توکن خودت را در config.php قرار بده
5️⃣	در ترمینال php bot.php را بزن
6️⃣	ربات را در یک گروه عضو کن و یک پیام بفرست
***************************************************************
ربات/
├── config.php      (تنظیمات مثل توکن ربات)
├── db.php          (کار با دیتابیس)
├── functions.php   (توابع کمکی)
└── bot.php         (خود ربات)
****************************************************************
قبل از اجرا، باید در phpMyAdmin یا ترمینال این دستور را اجرا کنی:

CREATE DATABASE IF NOT EXISTS news_db;
USE news_db;

CREATE TABLE IF NOT EXISTS news (
    id INT PRIMARY KEY AUTO_INCREMENT,
    message_text TEXT,
    channel_name VARCHAR(255),
    category VARCHAR(100),
    sender_name VARCHAR(255),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

****************************************************************
cd مسیر/پوشه/ربات
php bot.php
